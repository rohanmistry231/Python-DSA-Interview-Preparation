## String Algorithms

### Introduction
Efficient string matching.

### Types
- **KMP**: Prefix table for pattern search. O(n+m).
- **Rabin-Karp**: Hashing for pattern search. Average O(n+m).

### Advantages
- Fast searching.

### Disadvantages
- Preprocessing.

### Applications
- Text search.

### Time Complexities
- O(n+m).
