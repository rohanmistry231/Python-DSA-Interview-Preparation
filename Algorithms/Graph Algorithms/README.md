## Graph Algorithms

### Introduction
Algorithms for graph traversal and optimization.

### Types
- **Dijkstra**: Shortest path with weights.
- **BFS/DFS**: Traversal.
- **Topological Sort**: DAG ordering.

### Advantages
- Pathfinding, ordering.

### Disadvantages
- Complexity for large graphs.

### Applications
- Networks, dependencies.

### Time Complexities
- Dijkstra: O((V+E) log V).
