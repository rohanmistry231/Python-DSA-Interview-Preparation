## Dynamic Programming

### Introduction
DP solves complex problems by breaking into subproblems.

### Types
- **Memoization**: Top-down, cache results.
- **Tabulation**: Bottom-up, build table.

### Advantages
- Avoids recomputation.

### Disadvantages
- Space usage.

### Applications
- Fibonacci, knapsack.

### Time Complexities
- Polynomial.
